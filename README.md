Hastane Yonetim Sistemi for Java

TR
Ozellikler:

Guvenli Giris Sistemi: Sadece yoneticiler uygulamaya giris yapabilir.
Doktorlar Icin Sifre Yonetimi: Doktorlarin bilgileri sistemde saklanir ve her doktorun sifresi bulunur.
Hasta Kayit Sistemi: Yeni hastalar eklenebilir, mevcut hastalarin bilgileri guncellenebilir.
Teshis Kayit Sistemi: Doktorlar hastalar icin teshis bilgilerini kaydedebilir.


EN
Features:

-Secure Login System: Only administrators can log in to the application.
-Password Management for Doctors: Doctors' information is stored in the system and each doctor has a password.
-Patient Registration System: New patients can be added, information for existing patients can be updated.
-Diagnosis Registration System: Doctors can record diagnosis information for patients.
